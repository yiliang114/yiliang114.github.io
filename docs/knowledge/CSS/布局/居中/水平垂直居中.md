### 水平垂直全部居中

#### 1. 利用 vertical-align,text-align,inline-block 实现

```css
.parent {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
.child {
  display: inline-block;
}
```

#### 2. 利用绝对定位实现

```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  /* child 元素 x y 均移动 50% */
  transform: translate(-50%, -50%);
}
```

##### 让绝对定位的 div 居中

```css
div {
  position: absolute;
  width: 300px;
  height: 300px;
  margin: auto;
  top: 0;
  left: 0;
  bottom: 0;
  right: 0;
  background-color: pink; /* 方便看效果 */
}
```

#### 3. 利用 flex 实现

```css
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}
```
