### Node

基于 egg 来搭建的。 ts + egg ，为什么不用 koa2 因为 egg 不就是 koa 上封了一层么，自己能少做一点事情。

- 日志记录
- 日志、错误上报、错误邮件告警
- graph ql
- sequence 快速新建数据库表
- 快速创建路由并注册
- 权限系统
- api 网页调试工具， 不用每次都开 postman
  - 记录每次的值，直接点击即可
- 单元测试
