# 常见

找最长子字符串

```
string s1807 = "x1807";
for (int i = 0; i < s.length(); i++)
{
    for (int j = 1; j < 5; j++)
    {
        if (s[i] == s1807[j])
        {
            dp[i + 1][j] = max(dp[i][j - 1], dp[i][j]) + 1;
        }
        else
        {
            dp[i + 1][j] = dp[i][j];
        }
    }
}
```
